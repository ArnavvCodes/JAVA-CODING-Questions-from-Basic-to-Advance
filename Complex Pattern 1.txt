PROBLEM:
Write a Java program to print a right-aligned half pyramid of stars. The program should take the number of rows n as input.

Example Input:

5


Expected Output:

    *
   **
  ***
 ****
*****

CODE:
import java.util.Scanner;
public class pattern5 {
    public static void main(String[] arg){
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();       // rows count
        for(int i=1 ; i<=n ; i++){      //outer loop
            //inner loop : spaces
            for(int j=1 ; j<=n-i ; j++){
                System.out.print(" ");
            }
            // inner loop : star print
            for(int j=1 ; j<=i ; j++){
                System.out.print("*");
            }
            System.out.println();
        }
    }
}
