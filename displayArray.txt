PROBLEM:
Write a Java program to input an integer n and then input n integers into an array. Finally, display the entire array in a Array(readable) format only.

CODE:
import java.util.Scanner;
import java.util.Arrays;
public class array5 {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int[] ary = new int[n];
        for(int i=0;i<=(n-1);i++){
            ary[i]=sc.nextInt();
        }
        System.out.println(Arrays.toString(ary));
    }
}
