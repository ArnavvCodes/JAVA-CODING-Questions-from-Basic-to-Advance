PROBLEM:
Write a Java program to calculate the sum of all odd numbers from 1 to a given number n using a user-defined function.

CODE:
import java.util.Scanner;
public class func8 {
    public static int OddSum(int n){
        int sum = 0;
        for(int i=1;i<=n;i++){
            if(i%2!=0){
                sum = sum + i ;
            }
        }
        return sum;
    }
    public static void main(String[] arg){
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int sum = OddSum(n);
        System.out.println("SUM of odd numbers from 1 to "+n+"is:"+sum);
    }
}
