PROBLEM:
Write a Java program to print the elements of a 2D matrix in spiral order. The program should:
1.Take two integers n (number of rows) and m (number of columns) as input.
2.Take n Ã— m integers as input to fill the matrix.
3.Print the matrix elements in spiral order, starting from the top-left element and moving clockwise.

CODE:
import java.util.Scanner;
public class A2D2{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int m = sc.nextInt();
        int[][] matrix = new int[n][m];
        
        for(int i=0;i<n;i++){
            for(int j=0;j<m;j++){
                matrix[i][j] = sc.nextInt();
            }
        }
        System.out.println("The spiral Order Matrix is:");

        int rowstart = 0;
        int rowend=n-1;
        int colstart=0;
        int colend=m-1;

    while(rowstart<=rowend && colstart <= colend){
        for(int col=colstart;col<=colend;col++){
            System.out.print(matrix[rowstart][col]+" ");
        }
        rowstart++;
        for(int row=rowstart;row<=rowend;row++){
            System.out.print(matrix[row][colend]+" ");
        }
        colend--;
        for(int col=colend;col>=colstart;col--){
            System.out.print(matrix[rowend][col]+" ");
        }
        rowend--;
        for(int row=rowend;row>=rowstart;row--){
            System.out.print(matrix[row][colstart]+" ");
        }
        colstart--;
        System.out.println();
    }
  }
}
