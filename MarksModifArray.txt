PROBLEM:
Write a Java program to input ‘n’ numbers in an array and then input another number ‘m’. The program should find and display the index position of ‘m’ in the array if it exists.

CODE:
import java.util.Scanner;
public class array {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();    //for number of elements in the array
        int marks[] = new int[n];
        for(int i=0;i<=n-1;i++){
            marks[i] = sc.nextInt(); //for enter the data/elements in the array
        }
        int m=sc.nextInt();
        for(int i=0;i<=n-1;i++){  //for reading the array
            if(m==marks[i]){    //for comparing the user's value with array's elements
                System.out.println("indexing is:"+i);
            }
        }
    }
}
